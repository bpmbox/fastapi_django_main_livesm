```python

client = Client("http://localhost:7860")
result = client.predict(

    content="Hello World スクリプト作成",
    category="テスト",
    api_name="/create_test_prompt"

- `/simulate_execution` - 実行シミュレーション
- `/simulate_github_issue` - GitHub連携
- `/check_system_status` - システム状態確認
